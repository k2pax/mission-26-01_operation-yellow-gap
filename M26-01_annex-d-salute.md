# Annex D to OPORD 26-01, YELLOW GAP — SALUTE Spot Report

**Classification:** UNCLASSIFIED // PERSONAL USE ONLY

**Purpose:** A quick, structured format for radioing or texting back anything worth flagging — a hazard, wildlife encounter, washout, other party in distress, downed tree blocking a road, etc. Adapted from the standard military SALUTE format; fields relabeled for civilian backcountry use. Keep blank copies on hand — you'll want more than one across a 4-day patrol.

**Recipient default:** HERON 1 (primary), or Pisgah Ranger Station if it's something they should know about.

---

## SALUTE Report — Copy 1

| Field | Meaning | Report |
|---|---|---|
| **S** — Size | How many / how big (people, animals, trees down, extent of damage) | |
| **A** — Activity | What's happening / what you observed | |
| **L** — Location | Grid (USNG) or landmark + lat/long backup | |
| **U** — Unit/Description | Identifying detail — vehicle type/color, animal species, agency markings, party size | |
| **T** — Time | When observed (local + UTC if logging for radio) | |
| **E** — Equipment | Relevant gear/vehicles involved (theirs or yours) | |
